# Neovim

## Modes
`Esc` Normal
`i` Insert trước cursor
`a` Insert sau cursor
`v` Visual
`V` Visual Line
`:` Command

## Di chuyển
`hjkl` Trái/Xuống/Lên/Phải
`w/b` Từ tiếp/trước
`0/$` Đầu/cuối dòng
`gg/G` Đầu/cuối file
`C-d/u` Nửa trang xuống/lên
`{/}` Paragraph trước/sau
`%` Ngoặc đối diện

## Chỉnh sửa
`i/a` Insert trước/sau
`o/O` Dòng mới dưới/trên
`x` Xóa ký tự
`dd` Xóa dòng
`yy` Copy dòng
`p/P` Paste dưới/trên
`u` Undo
`C-r` Redo
`ciw` Đổi từ
`di"` Xóa trong ""
`5dd` Xóa 5 dòng

## Lưu & Thoát
`:w` Save
`:q` Quit
`:wq` Save & quit
`:q!` Quit ko save
`SPC w` Save
`SPC q` Quit

## Tìm kiếm
`SPC ff` Tìm file
`SPC fg` Grep text
`SPC fb` Buffers
`SPC fr` File gần đây
`SPC /` Tìm trong file
`/text` Tìm trong file
`n/N` Tiếp/trước

## LSP
`gd` Definition
`gr` References
`gi` Implementation
`K` Hover docs
`SPC ca` Code action
`SPC rn` Rename
`SPC D` Type definition
`SPC d` Diagnostics
`[d/]d` Diag trước/sau
`SPC cf` Format

## File & Window
`SPC e` File explorer
`S-H/S-L` Buffer trước/sau
`SPC bd` Đóng buffer
`:vsplit` Split dọc
`:split` Split ngang
`C-hjkl` Chuyển pane

## Tiện ích
`gcc` Comment dòng
`gc` Comment chọn
`C-\` Terminal float
`SPC th` Terminal ngang
`C-a` Select all
`SPC` Which-key

## Autocomplete
`Tab` Gợi ý tiếp
`S-Tab` Gợi ý trước
`Enter` Confirm
`C-Space` Mở menu
`C-e` Đóng menu
