# Tools

## Lazygit (`lg`)
`Space` Stage/unstage
`c` Commit
`p` Push
`P` Pull
`Enter` Xem diff
`[/]` Chuyển tab
`a` Stage all
`d` Discard
`b` Checkout branch
`n` New branch
`z` Undo
`?` Help
`q` Thoát

## fzf
`C-r` Tìm history
`C-t` Tìm file
`Alt+c` cd subfolder
`C-j/k` Lên/xuống
`Enter` Chọn
`Esc` Thoát
`Tab` Multi-select

## zoxide
`z foo` Nhảy tới "foo"
`z foo bar` Match cả hai
`zi` Interactive
`z -` Folder trước

## glow
`glow file.md` Render
`glow -p file.md` Pager

## bat (`cat`)
`cat file` Highlight
`catn file` + line numbers

## eza (`ls`)
`ls` List + icon
`ll` Chi tiết + hidden
`tree` Tree view
`tree -L 2` 2 cấp

## btop
`btop` Mở monitor
`f` Filter
`k` Kill process
`q` Thoát

## dust/duf/ncdu
`dust` Dung lượng folder
`duf` Dung lượng ổ đĩa
`ncdu` Interactive (xóa được)

## tldr
`tldr tar` Ví dụ tar
`tldr --update` Cập nhật

## thefuck
Gõ sai → `fuck` → tự sửa

## jq
`cat f.json | jq .` Format
`cat f.json | jq '.name'` Field
`cat f.json | jq '.items[]'` Array
`curl api | jq .` API
