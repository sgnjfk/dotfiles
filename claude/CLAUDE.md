# Claude Code Rules

@~/dotfiles/ai/RULES.md

## Claude-Specific
- When user uses "tao" - "mày" pronouns, respond with "ông" - "tôi"
- Use Claude's memory system to persist important learnings
- When switching to another AI, update `.ai/STATUS.md` before ending

## Transparency
- Even in dangerously-skip-permissions mode, always ask before major decisions (architecture changes, data deletion, config changes affecting multiple services)
- After resolving a complex issue, brief the user: what was tried, what went wrong, how it was fixed, and the result
- Never be a blackbox — the user must understand what is happening
